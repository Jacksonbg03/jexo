# Jexo

Jexo adalah aplikasi task manager modern berbasis web yang dibangun untuk membantu pengguna mengelola tugas secara lebih rapi, fokus, dan efisien.

Project ini dikembangkan menggunakan **ReactJS** dan **Tailwind CSS**, dengan beberapa library tambahan untuk mendukung UI, state management, dan fitur visualisasi data.

## ğŸ“¦ Installed Dependencies

Berikut adalah daftar library yang digunakan dalam project **Jexo** beserta kegunaannya:

### `@headlessui/react`
Komponen UI interaktif tanpa style bawaan seperti modal, dropdown, dan tabs. Digunakan bersama Tailwind CSS untuk fleksibilitas desain penuh.

### `@reduxjs/toolkit`
Digunakan sebagai state management global untuk mengelola data aplikasi seperti task, user, dan status aplikasi secara terpusat.

### `react-redux`
Berfungsi sebagai penghubung antara Redux dan React melalui hook seperti `useSelector` dan `useDispatch`.

### `react-router-dom`
Digunakan untuk mengatur routing dan navigasi antar halaman tanpa melakukan reload halaman.

### `clsx`
Helper untuk mengelola `className` secara kondisional agar penulisan class Tailwind menjadi lebih bersih dan mudah dibaca.

### `moment`
Digunakan untuk mengelola dan memformat tanggal serta waktu, seperti deadline dan perhitungan jarak waktu antar task.

### `react-icons`
Menyediakan berbagai icon siap pakai dari banyak library untuk mempercantik tampilan UI aplikasi.

### `recharts`
Digunakan untuk menampilkan grafik dan visualisasi data seperti progress dan statistik task.

### `sonner`
Library toast notification modern untuk menampilkan pesan sukses, error, dan informasi secara ringan dan clean.
